# 🎯 Real-Time Student Engagement Detection System

This project uses Computer Vision and AI techniques to monitor student engagement in real-time using a webcam.

## 🚀 Features
- Eye Aspect Ratio (EAR) for drowsiness detection
- Head pose tracking
- Hand gesture recognition
- Emotion detection (placeholder for ML model)
- Blink rate tracking
- Real-time GUI using Tkinter
- CSV logging for analysis
- Engagement classification:
  - Attentive
  - Distracted
  - Drowsy
  - Passive
  - Not Performing

## 🛠️ Tech Stack
- Python
- OpenCV
- MediaPipe
- NumPy
- Tkinter

## ▶️ How to Run

pip install -r requirements.txt
python main.py
